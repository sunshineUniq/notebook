nodejs实现网站的工作原理： 服务端动态生成html页面

安装express

```javascript
npm install -g express-generator@4
```

初始化项目

```javascript
express -e microblog
cd ./microblog
npm install
npm start // 启动项目
```

浏览器访问localhost:3000
